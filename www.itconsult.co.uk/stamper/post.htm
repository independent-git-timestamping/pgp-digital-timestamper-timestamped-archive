Proof of Posting Certificates

These are short cut instructions for using Stamper to produce Proof of Posting Certificates. Ideally they should be read in conjunction with the full instructions.

The email message should be prepared exactly as normal except that it should be addressed to post@stamper.itconsult.co.uk.

At the top of the body of the message add one line for each intended recipient:-

     X-Stamper-To: emailaddress

where emailaddress is the raw email address without any full names or the like.

Multiple recipients must have separate X-Stamper-To: lines. It is also important that the "X" must start in column 1. Should the message be a pgp file (for example clearsigned) the X-Stamper-To: directives must go outside the Begin/End PGP lines.

Example of Use

An example might be helpful of a message sent to post@stamper.itconsult.co.uk:-

From: matthew@itconsult.co.uk (Matthew Richardson)
To: post@stamper.itconsult.co.uk
Subject: Test Stamper Post
Date: Sun, 04 May 1997 09:46:30 GMT
Organization: I. T. Consultancy Limited, Jersey
Message-ID: <336c5aa9.61226569@itconsult.co.uk>
X-Mailer: Forte Agent 1.01/32.396

X-Stamper-To: test1@itconsult.co.uk
This is a test message
Testing 1 2 3 4

The following file is then delivered to the recipient:-

To: test1@itconsult.co.uk
From: matthew@itconsult.co.uk (Matthew Richardson)
Subject: Test Stamper Post
Date: Sun, 04 May 1997 09:46:30 GMT
Organization: I. T. Consultancy Limited, Jersey

X-Mailer: Forte Agent 1.01/32.396
MIME-Version: 1.0
Content-Type: text/plain; charset=us-ascii
Content-Transfer-Encoding: 7bit
