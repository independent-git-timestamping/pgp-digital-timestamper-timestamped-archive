Stamper List Server Instructions

list@stamper.itconsult.co.uk

The list server operates by reading lines in the BODY of any received message and processes each line as a command. It stops when it finds either a blank line or a line beginning with a dash ("-"). Multiple commands may be included within each message to the list server.

For each message (comprising one or more commands) it receives the list server will send a message in reply indicating each command it tried to execute and whether the command succeeded.

List Server Commands

The following commands may be sent to the list server. Optional parameters are given in square brackets.

JOIN listname [email-address]
This will join the email address given (or the email address of the sender if it is not given) to the given list. Thereafter, that email address will receive a copy of every message posted to the list. The word "SUBSCRIBE" may also be used instead of "JOIN".
LEAVE listname [email-address]
This will remove the email address given (or the email address of the sender if it is not given) from the given list. Thereafter, that email address will receive no further messages posted to the list. The word "UNSUBSCRIBE" may also be used instead of "LEAVE".
HELP listname
This will return whatever help file is available for the given list. If listname is omitted it will return these instructions.
DIR listname [email-address]
This will send a directory of available files for the given list to the email address given (or the email address of the sender if it is not given).
GET listname [email-address] filename
This will send the given file from the given list to the email address given (or the email address of the sender if it is not given).
DIGEST listname [email-address]
This will join the email address given (or the email address of the sender if it is not given) to the given list. Thereafter, that email address will receive a copy of every message posted to the list, but in the form of a single daily message containing all messages posted during the preceding day.

Lists Available

The list server operates the following lists, the names of which should be used as the "listname" parameter in the instructions above:-

STAMPER-ANNOUNCE
This list provides announcements (which will be occasional and brief) regarding the Stamper service. Users may JOIN or LEAVE the list only and may not post to it.
STAMPER-FILES
This list contains files relating to the Stamper service, which are automatically updated by Stamper. Users may only GET files or do a DIR to get a directory of files available.
STAMPER-WEEKLY
This list provides a weekly posting of the "Signatures of Signatures" provided by Stamper. Users may JOIN or LEAVE the list only and may not post to it.
STAMPER-DAILY
This list provides the daily summary which contains the detached signatures for the previous day. Users may JOIN or LEAVE the list only and may not post to it.
